Step 1: Build a personal syntax cheat sheet.

Run every script with bash -x during development.
bash-x myscript.sh

This prints every command before executing it with a + prefix. You see exactly where it breaks and what the variable values are at that moment. It's the bash debugger. 

Active recall every Sunday, 15 minutes

 Read: https://www.shellcheck.net/ — paste any script, it catches quoting errors instantly. Use ShellCheck on every script you write from now on.
